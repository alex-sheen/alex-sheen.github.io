3D Diver Interface

Unity, Meta Oculus
2022

For this project, we created an intuitive and usable HUD interface for scuba divers. While this assignment specified a couple required components such as a radio, teammate tracker, and depth meter, we were encouraged to add features we deemed necessary, and had free-reign in how we designed our interface. For example, my team anchored high priority buttons to the user's left wrist for fast access and designed a horizontal number pad interface for ease of use in 3D space and to safeguard against accidental misinputs. To test our interface without a real scuba suit, we simulated the diver's underwater experience in virtual reality using an Oculus headset and Unity.

The left wrist provides two buttons for the Radio and Crewmates Manager.

An SOS button is always accessible for emergencies.
The right wrist provides real-time updates on Altitude and Time Spent Underwater.
The Radio allows users to call crewmates.

We designed a horizontal numberpad ease of use in 3D, allowing for a vertical leeway, safeguarding against user error in mistakenly hitting adjacent buttons.
The Crewmates Manager provides crewmate call and crewmate tracking buttons. These buttons are placed vertically for ease of use in 3D, allowing for horizontal leeway.
The left button creates a green tracking arrow that points to the designated crewmate.

3D Diver Interface

Unity Game Engine
Meta Oculus Quest 2

For this project, we created an intuitive and usable interface for scuba divers. Our interface communicates features such as a radio to communicate with fellow crewmates, a tracker that points to teammates, and essential data such as depth and time spent underwater, To test our interface without a real scuba suit, we simulated the diver's underwater experience in virtual reality using an Oculus headset.

[pre-built functionality] - left pinch to move, breaststroke motion to swim.

The left wrist provides two buttons for the Radio and Crewmates Manager.

An SOS button is always accessible for emergencies.
The right wrist provides real-time updates on Altitude and Time Spent Underwater.
The Radio allows users to call crewmates. Crewmate numbers are conveniently located above the numberpad. The numberpad is oriented horizontally for ease of use in 3D, safeguarding against user error in mistakenly hitting adjacent buttons.
The Crewmates Manager provides two buttons. The right button allows for quick access to the crewmate call function. These buttons are placed on the sides of this screen for ease of use in 3D.
The left button creates a green tracking arrow that points to the designated crewmate.

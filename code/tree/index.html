Tree Space Colonization | Javascript

In this project, I implemented the tree generation algorithm featured in this paper: "Modeling Trees with a Space Colonization Algorithm". The space colonization algorithm starts at a root node and builds tree nodes into a cloud of "attraction points".

1. Each node finds the surrounding attraction points within the radius of influence 'di' and calculates the normalized vector from the node to attraction point.
2. Each node then grows a new node along the averaged vector from all of the normalized vectors.
3. Once all of the new nodes are created, a check is performed to remove any attraction point that is within kill distance 'dk'.
4. This process is repeated until no attraction point is within the distance 'di' of any node, or no attraction points remain.

Runions, A., Lane, B., and Prusinkiewicz, P. (2007). Modeling trees with a space colonization algorithm. In Proceedings of the Third Eurographics conference on Natural Phenomena, NPH’07, pages 63–70, Aire-laVille, Switzerland, Switzerland. Eurographics Association.

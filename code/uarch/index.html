Instruction-Level ARM Simulator
UChicago CMSC-22200 Computer Architecture, Aut 2021


In this course, I wrote a C program that is an instruction-level simulator for a subset of the ARMv8 instruction set. I implemented a 5-stage pipeline that handles branch cases, stall cases, and bypass cases. I also implemented branch prediction using an 8-bit global branch history register (GHR) and a branch target buffer (BTB). Lastly, I implemented a data cache and memory cache in my pipelined simulator (LRU, write-through, allocate-on-write).

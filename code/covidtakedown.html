covid taKedown

Visualization of K-means Clustering via Portable Clinic Placements for Covid Pandemic

Motorized Cubes - toio
Project Date - 2023


Our project was created during the Covid-19 pandemic to create an interactable system for optimizing the placement of temporary treatment centers or permanent hospitals to treat covid cases. The system uses a projector and toio robots to vizualize the underlying k-means clustering algorithm that drives the optimization. Our toio cubes with 3D printed shells represent the location of clinics, and they allow the user to physically interact with the algorithm.

The basic k-means clustering algorithm is non-deterministic, so the final output is affected by the initial centroid data points selection. Our interactable system provides an interface for this selection as the user can physically pick up and move each toio.

The algorithm will update appropriately as the user places or picks up different toios. One toio is designated as a control, which allows the user to scroll through the timeline by dragging it horizontally across the map. It can also be rotated to change the stylization of the centroid's regions. Additionally, it can be picked up and shaken to change the stylization of the data points.



Our team was one of teams selected to demo at the Chicago Museum of Science and Industry for their National Robot Block Party.

Visualization of K-means Clustering via Portable Clinic Placements for Covid Pandemic

toio robots, 3D printing, projector
2023

For CMSC 20380, Actuated User Interfaces and Technology, we developed an interactive system for optimizing the placement of temporary treatment centers or permanent hospitals for COVID-19 response.

The system integrates a projector and toio robots to visualize the underlying k-means clustering algorithm that drives the optimization process. Each toio cube, enclosed in a custom 3D-printed shell, represents a clinic location that users can manipulate to physically influence the algorithm’s behavior.

As users reposition the clinics on the board, the robots autonomously move to new locations, dynamically reconfiguring themselves to better serve patient distribution across the map.
Design Deep Dive

K-means clustering is a non-deterministic algorithm designed to partition a dataset into a predefined number of clusters. It operates through an iterative refinement process, gradually improving the positions of cluster centroids from an initial set of guesses.

Our interactive system complements this non-deterministic behavior by allowing users to physically pick up and move each toio, directly setting both the number of clusters and their initial centroid positions.

As users reposition the toios, the algorithm continuously updates in real time, reflecting how human interaction can guide and reshape computational processes.

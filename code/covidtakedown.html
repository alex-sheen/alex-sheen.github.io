Visualization of K-means Clustering via Portable Clinic Placements for Covid Pandemic

toio robots, 3D printing, projector
2023

For CMSC 20380, Actuated User Interfaces and Technology, created an interactable system for optimizing the placement of temporary treatment centers or permanent hospitals to treat covid cases.

The system uses a projector and toio robots to visualize the underlying k-means clustering algorithm that drives the optimization. Our toio cubes with 3D printed shells represent the location of clinics, and they allow the user to physically interact with and influence the algorithm.

As the user places the 3D printed clinics on the board, the clinics will physically move to organize themselves in a formation that will better service covid patients on the map.
Design Deep Dive

K-means clustering is a non-deterministic algorithm that aims to partition a dataset into a pre-defined number of clusters. The basic algorithm uses an iterative refinement process that slowly improves the position of the cluster centroids from an initial position guess.

Our interactable system complements the non-deterministic nature of the algorithm as our interface allows the user to physically pick up and move each toio to select the number of clusters and the initial centroid positions.

From there, the algorithm will continuously update as the user picks up or places different toios.
